ANG - Retail Sales Dashboard | Power BI

Project Overview

This project is an interactive Retail Sales Dashboard developed using Power BI.

The dashboard provides business insights related to:
- Total Sales
- Profit Analysis
- Profit Margin
- Product Performance
- Regional Sales Distribution
- Brand Performance
- Year-wise Sales Trends

The goal of this dashboard is to help businesses monitor sales performance, identify profitable products, and understand customer and regional trends.

---

Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Excel



Key KPIs

- Total Sales: 4.72 Billion
- Total Quantity Sold: 316K
- Total Profit: 873.55 Million
- Profit Margin: 18.49%



Dashboard Features

Interactive Filters
- Year-wise filter (2019–2024)
- Customer Segment slicer

Visualizations Used
- KPI Cards
- Bar Charts
- Donut Charts
- Monthly Sales Comparison
- Geographic Map Analysis
- Brand Performance Table

Business Analysis Included
- Top 10 Products by Sales
- Sales % by Store Type
- Sales % by Region
- State-wise Sales Distribution
- Sales vs Last Year Comparison
- Top Performing Brands



Key Insights

- Online stores contributed the highest sales percentage.
- South region generated the highest sales contribution.
- Products like Desk and Washing Machine were among the top-selling products.
- Some brands generated strong profit despite lower quantity sales.
- Monthly sales remained relatively stable with growth visible in later months.
